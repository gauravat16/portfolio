How to use Lookup

Lookup Button : For Android verison Marshmallow (6) and above

  1. Tap and select a word
  2. Tap on 3 vertical dots
  3. Tap on Lookup button

Lookup on Copy : Exclusively for Android version Oreo (8) and below

  1. Tap and select a word
  2. Tap on 3 vertical dots
  3. Tap on Copy button
